# LGBTQ-Hate-Speech-Detection-System
A hate speech detection system using BERT (Bidirectional Encoder Representations from Transformers) to identify and classify anti-LGBTQ+ cyberbullying and hate speech. This project achieves 90.7% accuracy with sophisticated deep learning techniques.
Overview
This project implements a BERT-based binary classifier for detecting hate speech targeting the LGBTQ+ community. The system uses transfer learning from Google's pre-trained BERT model, fine-tuned on a specialized dataset of anti-LGBTQ+ content.
Why This Matters
Online hate speech targeting LGBTQ+ individuals is a pervasive problem. This tool aims to:

Protect vulnerable communities from online harassment
Automate content moderation at scale
Provide insights into hate speech patterns
Assist researchers studying cyberbullying

Key Highlights

✅ 90.7% Test Accuracy on real-world data
✅ 0.84 F1-Score demonstrating balanced performance
✅ Real-time Analysis with confidence scoring
✅ Batch Processing for large-scale content moderation
✅ User-Friendly Web Interface built with Streamlit
